# homebrew-tap

Homebrew tap for [alexrao2000/git-tools](https://github.com/alexrao2000/git-tools).

```bash
brew tap alexrao2000/tap        # once
brew trust alexrao2000/tap      # once — Homebrew gates third-party taps
brew install gtools             # thereafter just: brew upgrade gtools
```

Installs the `git sweep`, `git sync`, `git pr`, and `git done` subcommands.
